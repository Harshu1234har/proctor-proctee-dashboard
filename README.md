# Proctor–Proctee Dashboard

This is a college-level web application designed to manage the
Proctor–Proctee system.

## Roles
- **Admin**
  - Create, edit, delete proctors
  - View all proctors and proctees
- **Proctor**
  - Create, edit, delete proctees (students)
- **Proctee**
  - Student academic details

## Features
- Role-based access control
- Proctor management
- Proctee management with USN, branch, year, email
- Clean and simple UI

## Technologies Used
- HTML
- CSS
- JavaScript

College Mini Project / Web Technology Project
